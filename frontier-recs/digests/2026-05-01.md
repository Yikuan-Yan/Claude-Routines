#### 为什么选这篇

湍流是物理学中最顽固的未解问题之一，正是因为描述它的非线性方程在一般情况下完全不可积。这篇论文在一个精确可积的框架内——可积湍流（integrable turbulence）——构造了波-均流相互作用的**精确解析解**。精确解不依赖闭合假设或多尺度近似，而是从可积结构的第一性原理直接导出。即使这是非线性薛定谔方程（NLS）等特殊情形，它所揭示的机制和标度行为将为更一般的非线性波-湍流耦合理论提供无可替代的理论基准。

#### 问题背景

波-均流相互作用（wave-mean field interaction）是地球物理流体动力学的核心问题：海浪通过 Stokes 漂移驱动表层洋流，大气重力波通过辐射应力（radiation stress）修改背景经向环流。耦合方程组（波动方程 + 均流方程）在一般情况下无解析解，通常依赖 WKB 方法或 Whitham 多尺度展开，引入了"波包轮廓缓变"等近似条件。

可积湍流则不同。在完全可积 PDE（NLS、KdV、Toda 晶格）中，系统拥有无穷多守恒量，通过逆散射变换（IST）可对孤子气（soliton gas）的热力学进行精确描述（El 2003, Zakharov 1971）。波-均流耦合纳入这个框架之前从未被解析处理：问题在于均流分量的演化方程如何从可积结构中自洽导出，以及它是否保持可积性。

#### 核心结果与物理图像

论文在可积 PDE 框架下推导了有限振幅波谱（辐射带）与相干均流之间耦合动力学的精确方程组，证明该耦合系统在特定条件下仍可通过 IST 精确求解。

关键结论：

1. **精确标度律**：在长波极限下，均流速度服从精确幂律演化 U(t) ∝ t^{−α}，指数 α 由初始孤子气的谱密度函数决定，与动力学细节无关——这是可积守恒律赋予的普适性。

2. **封闭均流反作用**：波列对均流的辐射应力在可积情形下有精确封闭表达式，避免了传统 Bretherton-McIntyre 框架中依赖波包缓变假设的近似。均流的动量变化速率可表示为孤子气谱函数的线性泛函，直接可计算。

3. **相图结构**：参数空间中出现波主导、均流主导和混合演化三种 regime 的边界，边界位置由初始波谱的非线性参数（孤子密度与辐射能量的比值）精确确定。

物理图像：在可积湍流中，孤子携带固定的守恒动量；当波谱演化并将能量传递给均流时，守恒律约束使得总过程像一个精确的"能量重分配机器"——没有耗散，也没有混沌的不可预测性。这与真实 Navier-Stokes 湍流的本质区别恰好使精确解成为可能。

#### 批判与展望

可积性是极强的约束——真实湍流不是可积的，实际系统存在各种可积性破坏（integrability-breaking）效应（粘性、非均匀介质、多维效应）。从可积极限到弱可积性破坏的过渡如何定量描述，是直接的后续问题。同时，NLS 可积湍流已在光纤光学实验（Suret 等 2016, Tikan 等 2022）和浅水波槽中被部分实现；本文的均流预测是否可在这些受控实验中验证？若能实现，将是首个波-均流可积理论的精确实验检验。

#### 为什么选这篇

共形不变性（conformal invariance）是二维临界系统的标志性特征：它比尺度不变性更强，要求关联函数在保角变换（angle-preserving maps）下严格协变，从而固定关联函数的具体代数形式，而不仅仅是幂律指数。这种对称性以前只在磁相变（Ising 临界点）、随机游走、多孔介质渗流等平衡统计力学系统中被精确验证。这篇论文的结论极为大胆：在远离热平衡的活细胞单层中，速度涨落满足共形场论（CFT）的 Ward 恒等式，且共形数据（中心荷、标度维度）与已知的离散 CFT 类吻合。如果成立，它意味着生命物质在集体动力学层面"涌现"出了等价于平衡临界系统的精确对称性——而非只是粗糙的"像临界"行为。

#### 问题背景

临界性假说（criticality hypothesis）认为生物系统通过演化调谐至相变临界点附近，以最大化信息处理能力和动态响应范围（Bialek 等 2012）。但"临界"通常仅以幂律关联和临界慢化等特征识别，而这些特征不唯一——许多非临界系统也可呈现幂律。共形不变性是一个更严格的判据：它不仅要求幂律，还要求 n 点关联函数满足共形 Ward 恒等式以及共形块（conformal block）分解的特定系数结构。这两类预测在统计意义上可检验，为区分"真正的 CFT 固定点"和"仅有幂律的非临界系统"提供了定量工具。

活性物质（active matter）的流体力学描述已知在某些条件下具有吸引子，但吸引子是否恰好对应一个 CFT 不动点，此前从无实验支持。

#### 核心结果与物理图像

论文使用上皮细胞单层（epithelial monolayer）的高分辨率速度场数据，系统检验了 2D CFT 的核心预测：

1. **Ward 恒等式验证**：2 点和 3 点速度涨落关联函数（两阶张量）在系列角度和距离采样后，与 CFT Ward 恒等式的预测在 ±10% 以内符合，显著优于仅有尺度不变性时的预测。

2. **中心荷提取**：从关联数据拟合的有效中心荷 c_eff 收敛至接近 c = 1（自由玻色子场），而非随机中间值；这一结果在至少两种不同上皮细胞系（MDCK、HaCaT）中重现，提示"普适性"在于与细胞类型无关的共形类别。

3. **活性驱动的"涌现临界"**：系统处于稳态增殖（细胞分裂与挤出保持稳定总数）条件下；论文论证这一驱动-耗散平衡对应一个非平衡不动点，其有效场论恰好具有 c = 1 共形对称性。

物理图像：细胞分裂产生局部膨胀，细胞挤出（extrusion）产生局部收缩，两者的竞争在中等时空尺度上建立起一个"活性湍流"稳态，这个稳态与自由标量场的 CFT 不动点处于同一普适类——尽管微观机制截然不同。类比：液体-气体临界点和某些聚合物系统属于同一普适类（Ising），但微观物理完全不同。

#### 批判与展望

最重要的方法论问题：四点关联函数的共形块分解需要极高的统计质量，生物实验数据的信噪比是否真的足够？现有结果基于二点和三点函数，若能进一步验证四点函数，结论将大幅加强。另外，c_eff ≈ 1 与自由玻色子 CFT 吻合是有趣的，但究竟是偶然巧合还是深刻机制？在非平衡系统中，"中心荷"不再有 c-theorem 约束（即它在重整化群流中不一定单调减少），c = 1 的意义需要新的理论框架解释。自然的后续是：用类似方法分析神经系统（神经元活动场）和发育中的组织，检验 c = 1 的普适性是否跨越不同生物系统——这将把"活性 CFT"确立为描述集体生物动力学的新范式。

### A3. Long-Range Order in a Strictly Short-Range Quasi-2D XY Model

#### 为什么选这篇

Mermin-Wagner 定理（1966）是统计力学的基石：在一维或二维系统中，短程相互作用的连续对称群不能在有限温度下自发破缺。六十年来，该定理被一再引用，也被实验反复"违反"——但仔细检查后，每次"违反"都对应定理的某个技术前提（真正的无限大热力学系统、严格短程相互作用）被悄悄放宽。这篇论文做了一件严谨的事：它精确界定了在**严格短程**相互作用下，**准二维**（多层）XY 模型何时产生真正的长程序，给出了临界温度 T_c 对层间耦合常数 J⊥ 的精确标度关系，并定量回答了"多薄才算二维"这个真实材料中无法回避的问题。

#### 问题背景

XY 模型中，二维单层在有限温度的低温相具有幂律关联（BKT 相），但不存在真正的长程序（⟨m⟩ = 0）。真正长程序（⟨m⟩ > 0）只在 T = 0 出现，且受热涨落破坏。

"准二维"（quasi-2D）意味着系统在 z 方向有有限层数 N_z，而非单一原子层。严格来说，任何 N_z ≥ 2 的分层系统都是三维的，Mermin-Wagner 定理并不直接禁止其长程序。但实验者和理论家都知道，在 J⊥/J‖ → 0 时，分层系统应该越来越像"纯二维"——问题是：这个极限有多"缓慢"？T_c 是否随 J⊥ → 0 连续趋于零，还是在某个有限 J⊥ 处突然消失？以及，T_c 的精确函数形式是什么？这个问题对于二维磁性材料（CrI₃、Fe₃GeTe₂ 双层、MnBi₂Te₄ 多层）的磁序分析具有直接应用价值。

#### 核心结果与物理图像

论文对 N_z = 2, 4, 8 层的 XY 模型（面内和层间均为最近邻耦合，严格无长程成分）进行蒙特卡洛模拟，辅以有效场论分析，得出：

1. **有限温度相变存在**：对于任意 N_z ≥ 2 且 J⊥ > 0（无论多小），在足够低的温度下总存在真正的铁磁长程序，相变温度：

$$T_c(N_z, J_{\perp}) \approx \frac{T_c^{3D}}{1 + A \ln(J_{\|}/J_{\perp})}$$

其中 A 为与 N_z 相关的常数，$T_c^{3D}$ 为各向同性三维极限的临界温度。

2. **对数标度**：$T_c$ 随 J⊥/J‖ → 0 以**对数**方式趋于零，而非幂律。这意味着即使层间耦合比面内耦合弱 10³ 倍，T_c 仍约为三维值的 25%——对数的"缓慢衰减"使准二维磁体比直觉预期更容易保持磁序。

3. **BKT-to-Ising 交叉**：对于 N_z = 2 层，相变在 J⊥ 很弱时具有 BKT 特征（无长程序突变），但随 J⊥ 增大，交叉为真正的 Ising 型有限温度相变（⟨m⟩ 在 T_c 处突变为零）。这个交叉本身的位置也给出精确条件。

物理图像：层间耦合 J⊥ 作为一个弱"对称破缺场"，抑制了二维系统中发散的相位涨落（Goldstone modes）。即使 J⊥ 远小于 J‖，系统的最低频率涨落模式不再是纯二维的 ω ∝ k²，而是各向异性的 ω² ∝ k²_‖ J‖ + k²_⊥ J⊥——后者的存在使得相位涨落的红外发散被截断，从而允许长程序。

#### 批判与展望

论文的结论在数学上清晰：任何有限层系统都是 3D，Mermin-Wagner 从不禁止其长程序。论文的贡献是给出了 T_c 的**精确标度形式**，填补了"定性知道有序"和"定量计算何时有序"之间的空白。直接应用：对于 CrI₃ 双层（$J_{\perp}/J_{\|} \approx 0.01$），代入论文公式可预测 T_c 的数量级，与实验测量值（约 45 K）对比是直接的验证。另一个开放问题：在量子 XY 模型（即 XXZ 模型的 XY 极限）中，量子涨落是否会改变这一对数标度关系？这将有助于理解磁性原子层中量子效应对磁序的竞争。

非厄米系统的**例外点**（exceptional point, EP）是本征值和本征矢量同时简并的奇异点，此处哈密顿矩阵退化为 Jordan 块形式。EP 附近的物理独特：本征值对参数扰动的响应为 $\varepsilon^{1/n}$（n 阶 EP），而非通常的 $\varepsilon^1$，这被用于提出超灵敏传感方案。然而，是否在 EP 处是一个二值判断，无法描述系统"距 EP 多远"或"EP 的强度如何"。

本文引入"例外亏量"（exceptional deficiency, ED）——一个连续的实数量，刻画非厄米矩阵的 Jordan 结构偏离正常（完全可对角化）哈密顿量的程度。ED = 0 对应普通非厄米系统（无简并），ED = 1 对应完整 n 阶 EP。通过 ED 可以定量排序不同非厄米系统的"EP 程度"，并预测其灵敏度增强因子的实际量级（包含 ED 依赖的修正项）。实验验证在光子晶格和声学超构材料中完成：不同 ED 的样品表现出与理论预测吻合的传输响应和灵敏度标度。这为非厄米物理从定性描述（"有没有 EP"）向定量分类迈出了一步。

大涡模拟（LES）的核心瓶颈是**亚格尺度（SGS）应力闭合**：网格以下的湍流微结构如何影响可分辨大尺度场？传统 Smagorinsky 模型给出简单耗散，但在复杂壁面边界层中严重低估 back-scatter（小尺度向大尺度的逆向能量传递）。数据驱动 SGS 模型（用 DNS 数据训练的神经网络）可以改进精度，但通常对训练分辨率/雷诺数过拟合。

LESnets 用**神经算子**（neural operator）框架（基于 FNO/DeepONet 思路）学习从粗粒化速度场到 SGS 应力场的映射，同时在损失函数中嵌入动量守恒约束（physics-informed 部分）。在 $Re_\tau = 180$ 至 5200 的壁面湍流通道流中，LESnets 的 SGS 应力预测误差比纯数据驱动基线低约 30–40%，且在训练之外的网格尺寸上泛化显著优于 FNO 基线。物理约束的加入同时防止了能量堆积（energy pile-up）这一神经网络 SGS 模型的常见病理现象。对计算流体力学社区，这一结果表明 physics-informed neural operators 正在从学术演示转变为可部署的工程湍流建模工具。
