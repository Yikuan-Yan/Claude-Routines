# Day 3: 为什么彩虹是弧形的，而且你永远走不到它脚下？

## 直觉入口

雨停了。太阳从西边的云缝里钻出来，你转过身，背对着它，于是那道弧就在东边的天空上。红在外，紫在内，横跨半个天穹。你本能地想走过去看看它"落"在哪——左脚那棵树的后面？田野的尽头？你迈开步子。

然后你会发现一件非常奇怪的事：彩虹跟着你走。

不是跟着你的眼睛转，而是整体平移。你向它走一百米，它退一百米。开车追它，它以车速后退。从飞机上往下看，彩虹甚至可以是一个完整的圆圈。它显然在那里——肉眼可见，相机可拍——但它又显然不在任何地方。

**这不是一个物体，而是一个方向。** 搞清楚这一点，三个谜题同时解开：为什么是 42° 这个精确角度？为什么是弧形（而不是一片、一块、一条直线）？为什么你走不到它脚下？

## 为什么直觉是错的（或不够的）

最流行的直觉解释是："水滴像小棱镜，把白光分解成彩虹。" 这句话不算错，但它回答了错误的问题。棱镜确实能分光，可棱镜分光得到的是一条光谱带，方向取决于棱镜的摆放。如果水滴只是小棱镜，彩虹的位置应该依赖于每个水滴的朝向——而水滴是圆的，在空气里随机下落，没有"朝向"可言。随机朝向的棱镜群只会把阳光均匀打散，给你一片惨白的雾光，而不是一道 42° 处的锐利弧。

第二个常见误解是关于 Alexander's dark band——主虹（42°）和副虹（51°）之间那条明显偏暗的天空带。很多科普文章写"这是因为光在水滴内部发生了全内反射"。这是错的。水的临界角 θ_c = sin⁻¹(1/1.33) ≈ 48.6°，而彩虹成因光线在水滴背面的入射角约为 40°——远低于临界角，**根本不发生全内反射**。每次碰到内表面，只有约 4% 的光反射回来，其余都漏出去了。彩虹本身就是"大部分光漏光后剩下的那一小撮"的集中显示。

所以正确的追问是：为什么一颗圆水滴——没有方向性，只发生部分反射——会让光在一个精确的 42° 方向上集中？为什么这个集中对波长敏感？

## 物理图像

1637 年，Descartes 在《方法论》的附录里做了一件今天看来非常现代的事：他不去问"彩虹为什么有颜色"，而是追踪一条光线在一个球形水滴里的命运。他问的是纯粹的几何问题——光从哪里进、从哪里出、往哪个方向走。

设一束平行阳光打到球形水滴上，入射角为 i（相对于入射点的法线，也就是该点到球心的连线）。光折射进入水滴，折射角 r 满足 Snell 定律 sin i = n sin r。在水滴背面发生一次（部分）内反射，然后从前表面再折射出去。追踪几何，出射光相对于原入射方向的总偏折角是

$$D(i) = 180° + 2i - 4r.$$

这条公式不起眼，但它是整个彩虹现象的骨架。**关键不在这个函数本身，而在它的导数。**

想象你把一束平行光的不同光线——打在水滴上缘、中部、接近边缘——都追踪一遍。每条光线的 i 不同，所以 D(i) 也不同。一般而言，入射角变化 Δi 会导致出射方向变化 ΔD。但 D(i) 作为一个函数，在某个特定的 i* 处有**极小值**：dD/di = 0。在这个极小点附近，D 对 i 几乎不变——也就是说，入射角在 i* 附近一小片区域里的光线，全部几乎朝同一个方向出射。这就是光的几何集中。

这是彩虹最深的物理：**焦散 (caustic) 是极值点的奇迹**。不是因为这个方向反射更多，不是因为这个方向不吸收——纯粹是因为 D(i) 的极值让大片入射光线"折叠"到同一个出射方向上。其他方向也有光出来，但稀散；只有极值方向，光强陡增。

从 dD/di = 2 - 4(dr/di) = 0 得到 dr/di = 1/2；对 Snell 定律 sin i = n sin r 两边微分，cos i di = n cos r dr，所以 cos i = (n/2) cos r。结合 Snell，经过一些三角代换，最终得到一个漂亮的闭式：

$$\cos^2 i^* = \frac{n^2 - 1}{3}.$$

水的 n ≈ 4/3，所以 cos²i* = (16/9 - 1)/3 = 7/27 ≈ 0.259，i* ≈ 59.4°。对应的 r* ≈ 40.2°，D_min ≈ 138°。彩虹方向（相对 antisolar point，也就是你影子头顶的方向）= 180° - 138° = **42°**。

这个 42° 是纯粹几何的产物。它不依赖水滴大小，只依赖 n。

现在回到三个谜题：

**为什么是弧形？** 水滴是球形的，关于入射光方向对称。所以对每一颗水滴来说，42° 不是一个方向，而是一个以 antisolar 方向为轴的**锥面**——锥顶半角 42°。空中无数颗水滴，每一颗都把它接收到的阳光散射成这样一个 42° 锥面。你只能看到落在"你的 42° 锥"上的那些水滴散出的光——这就是一条圆弧（被地平线切一半）。从飞机上、在瀑布边，地平线不挡，你就能看到整圆。

**为什么走不到它脚下？** 这个圆弧的轴线是你自己的 antisolar 方向——过你的眼睛、与太阳光相反的那条射线。你动一步，antisolar 方向跟着你动，整个 42° 锥面跟着你平移。彩虹不是挂在空间里的物体，而是**附着在观察者身上的方向图案**。两个人站在一起看同一道彩虹，其实看到的是两道几何上不同的彩虹——由两组不同的水滴散射出来，只不过方向重合得几乎完美。

**为什么有颜色？** n 对波长 λ 有微弱依赖（色散）。水在可见光中 n 从红光的 1.332 到紫光的 1.343，差 0.011，约 0.8%。这点微小差异通过 cos²i* = (n²-1)/3 放大到几何上——红光的 i* 略大、r* 略小、D_min 略小、彩虹角略大。结果：红色在 42.2°，紫色在 40.6°，差 **约 1.6°**。彩虹的可见"宽度"就是这个 1.6°——比太阳直径（0.5°）大三倍，所以颜色肉眼可分辨。

副虹（secondary rainbow）是光在水滴内部经历两次内反射后出来，偏折积累一多 180° - 2r 的折返，重新做极值分析得到 cos²i* = (n²-1)/8，对应彩虹角 51°。两次反射把颜色顺序翻过来——副虹里红在内、紫在外。主虹外侧（> 42°）没有来自主虹机制的集中光，副虹内侧（< 51°）也没有来自副虹机制的集中光。两者中间这条 42°–51° 的环，**既不是 TIR 导致的暗，也不是吸收**，而是单纯的几何真空：没有哪种机制把光集中到这里。这就是 Alexander's dark band。

## 定量估算

把几何做扎实。水 n = 4/3（可见光平均值）。

**第一步：** 极值条件 cos²i* = (n² - 1)/3 = (16/9 - 1)/3 = 7/27。取平方根，cos i* ≈ 0.509，i* ≈ 59.4°。

**第二步：** Snell 定律给出 r*：sin r* = sin(59.4°)/n = 0.861/1.333 ≈ 0.646，r* ≈ 40.2°。

**第三步：** 代回 D = 180° + 2i - 4r = 180° + 118.8° - 160.8° = **138°**。彩虹角 = 180° - 138° = **42°**。三步算出来，没有查表。

**色散的量级估算。** 可见光谱从 700 nm（红）到 400 nm（紫），n 从 1.332 变到 1.343，Δn ≈ 0.011。对极值角 i* 求微分：2 cos i* (-sin i*) di* / (1) = (2n/3) dn，即 di*/dn = -(n/3) / (cos i* sin i*)。数值代入：di*/dn ≈ -(1.333/3) / (0.509 × 0.861) ≈ -1.01 rad，所以 Δi* ≈ -0.011 rad ≈ -0.63°。这是 i* 的变化；真正决定彩虹角的是 D_min 的变化。D = 180 + 2i - 4r，所以 ΔD = 2Δi - 4Δr。从 Snell 微分出 Δr，算到最后 ΔD ≈ -1.7°（红到紫 D_min 变小 1.7°，对应彩虹角从 42.2° 变到 40.6°）。**观测值 1.6°**——估算精度到 5% 以内，几何骨架经得起推敲。

**亮度量级。** 进入水滴的光，在第一次入射面（59° 入射角，非垂直）Fresnel 反射掉约 6%，剩下 94% 折射进入；在水滴背面的内表面，入射角 40°（远低于临界角 48.6°），只有约 **5–6% 被反射回来**，其余 ~94% 漏掉；再从前表面折射出来，又损失 ~6%。主虹总效率 ≈ 0.94 × 0.055 × 0.94 ≈ **5%**。也就是说，你看到的彩虹只承载了 20 分之 1 的入射光能。这也是为什么彩虹只在阳光充足且背景偏暗（深色云层）时显眼——信噪比不够，它就被天空的散射光淹没。

## 反事实探索——如果水没有色散会怎样？

色散是指 n(λ) 对波长的依赖。水的 Δn ≈ 0.011 across visible。这 0.011 是一个微不足道的数字——比水的绝对折射率小两个量级。但我们已经看到，它被几何放大成 1.6° 的彩虹宽度，红紫能分开。

现在做一个思想实验：**把色散关掉**。设想一种假想的"水"，对所有可见光波长 n 精确等于 4/3。其他一切不变——球形水滴，部分内反射，D(i) 几何不变。

第一步会发生的事：彩虹依然存在。极值条件 cos²i* = 7/27 不依赖 λ，所以所有颜色在同一个 42° 处集中。彩虹的亮度、位置、圆弧形状都保留。

第二步：**颜色消失**。所有 λ 叠在同一个角度上——你看到的是一道纯白的弧。红色没有分出来，紫色没有分出来，它们精确重合。这不是"没有光"，而是"光都挤在一个角度"。

第三步：弧会变窄。实际彩虹的 1.6° 角宽度几乎全来自色散。没有色散，理论上彩虹会压缩到一个无限薄的几何亮线——但这里就是经典几何光学破功的地方。水滴尺度 (~1 mm) 与可见光波长 (~500 nm) 比值约 2000，虽然大，但不是无穷。在极值方向附近，干涉效应把理想几何线展宽成一个有限宽度的强度剖面——这就是 Airy (1838) 的波动光学处理，结果是一个 **Airy function** 的强度图样。

**第四步：这个反事实接近真实的 fog bow。** 雾滴直径通常 < 100 μm，比雨滴小一个量级。波长与尺度之比增大，干涉效应显著；色散被波动展宽淹没——结果是一道几乎纯白的"雾虹"（也叫"白虹"），出现在雾天太阳背后的方向，角度还在 42° 附近，但颜色全无。小水滴抹平色散效应，这是几何光学与波动光学的过渡。

再看一个对偶的反事实：**把水换成玻璃球**（n = 1.5）。cos²i* = (2.25-1)/3 = 0.417，i* ≈ 49.8°，r* ≈ 30.6°，D_min ≈ 157°，"虹角" = 23°。如果空中下的是玻璃球而非水滴，彩虹会出现在距 antisolar point 只有 23° 的位置——一个比真实彩虹紧凑得多的圆，几乎环绕在你影子头顶周围。它会让整个天空的彩虹位置重新布置一次。

这个数字 23° 不是凭空的巧合——它接近**冰晶造成的 22° halo** 的角度。但 22° halo 的物理机制完全不同：不是球形水滴的两次折射一次内反射，而是六角形冰晶的两个相差 60° 的晶面折射。机制不同、几何不同，只是数值上偶然靠近——物理里这种偶合提醒我们，同一个角度可以承载完全不同的故事。

## 跨现象连接

**从 Day 2 的散射过渡。** 昨天讲大气分子把蓝光散得满天都是，Rayleigh 散射的 λ⁻⁴ 是一种弥漫的、无方向的波长选择。今天讲的彩虹是相反的极端：高度方向化（精确 42°）、高度局域化（一个薄圆弧）、但同样是波长选择。两种光-物质相互作用共用一个前提——光的频率远离物质共振频率。不同的是结构尺度：分子（~Å）远小于 λ → Rayleigh；雨滴（~mm）远大于 λ → 几何光学 + 色散。尺度跨了 7 个量级，物理图像彻底换了一种语言。

**Supernumerary arcs 是几何光学的波动修正。** 细心观察会看到，主虹内侧紧贴着几条淡粉-淡绿交替的细纹，被称作 supernumerary bows。Descartes 的几何光学完全无法解释——它们是波动干涉产生的。Thomas Young 1804 年意识到："光线稍不同的两条路径出射方向相同"，这是双缝干涉的场景。Airy 1838 年给出完整理论：在极值方向附近，强度不是一个简单峰，而是一个 **Airy function** 的震荡尾巴。Supernumerary 峰的位置精确取决于水滴直径——只有当水滴尺寸足够均匀（直径 0.5–1 mm 左右）时才能看到清晰的 supernumeraries。雨如果太粗或太细，它们就模糊成普通的彩虹内侧。这是一个罕见的例子，**地面上肉眼可见的干涉图样**——光的波动性在日常现象里直接显形。

**Glory 是另一条路。** 你在飞机上俯视云层，会看到飞机影子周围一圈同心彩色光环，称为 glory。它的机制与彩虹完全不同：光在 antisolar 方向（精确 180°）后向散射并形成干涉环。几何光学里 180° 不是极值点，所以 Descartes 式分析给不出 glory——完整处理需要 Mie 散射理论，van de Hulst 1947 年提出的 surface wave 图像给出了一种直观解释（光沿水滴"赤道"爬行后跳出），但至今 glory 的精确机制仍涉及边缘射线、表面波和 Mie 共振几种贡献的叠加，是 Mie 理论里的深水区。彩虹、fog bow、glory——都是小水滴与阳光的合作，却各自对应完全不同的物理图像。

**Day 4 预告：肥皂泡的薄膜干涉。** 彩虹靠色散（dn/dλ）分光；肥皂泡靠**几何路径差**分光——薄膜上下表面反射光的光程差随膜厚与波长变化，不同波长在不同厚度处建设性干涉。两种都能把白光分解成颜色，但一个依赖介质对波长的响应 n(λ)，另一个依赖波长相对几何尺寸的比例关系 d/λ。同样是彩色的弧（或泡面上的彩色条纹），底层机制隔了整整一门学科：色散属于介质响应（材料物理），薄膜干涉属于波动叠加（基础波动力学）。

彩虹的终极教训是：**不要把光学现象想成"东西"，要想成"条件"。** 42° 不是空间里的一个地方，而是一个几何条件——过你的眼睛、与 antisolar 方向夹角 42° 的锥面。只要条件满足，任何水滴都能参与；条件不满足，再多水滴也无关。你走不到彩虹脚下，不是因为它"跑"，而是因为它从一开始就不是那种可以"去"的东西。
