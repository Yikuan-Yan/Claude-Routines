# Day 3: 为什么彩虹是弧形的，而且你永远走不到它脚下？

## 直觉入口

雨停了。太阳从西边的云缝里钻出来，你转过身，背对着它，于是那道弧就在东边的天空上。红在外，紫在内，横跨半个天穹。你本能地想走过去看看它"落"在哪——左脚那棵树的后面？田野的尽头？你迈开步子。

然后你会发现一件非常奇怪的事：彩虹跟着你走。

不是跟着你的眼睛转，而是整体平移。你向它走一百米，它退一百米。开车追它，它以车速后退。从飞机上往下看，彩虹甚至可以是一个完整的圆圈。它显然在那里——肉眼可见，相机可拍——但它又显然不在任何地方。

**这不是一个物体，而是一个方向。** 搞清楚这一点，三个谜题同时解开：为什么是 42° 这个精确角度？为什么是弧形（而不是一片、一块、一条直线）？为什么你走不到它脚下？

## 为什么直觉是错的（或不够的）

最流行的直觉解释是："水滴像小棱镜，把白光分解成彩虹。" 这句话不算错，但它回答了错误的问题。棱镜确实能分光，可棱镜分光得到的是一条光谱带，方向取决于棱镜的摆放。如果水滴只是小棱镜，彩虹的位置应该依赖于每个水滴的朝向——而水滴是圆的，在空气里随机下落，没有"朝向"可言。随机朝向的棱镜群只会把阳光均匀打散，给你一片惨白的雾光，而不是一道 42° 处的锐利弧。

第二个常见误解是关于 Alexander's dark band——主虹（42°）和副虹（51°）之间那条明显偏暗的天空带。很多科普文章写"这是因为光在水滴内部发生了全内反射"。这是错的。水的临界角 θ_c = sin⁻¹(1/1.33) ≈ 48.6°，而彩虹成因光线在水滴背面的入射角约为 40°——远低于临界角，**根本不发生全内反射**。每次碰到内表面，只有约 4% 的光反射回来，其余都漏出去了。彩虹本身就是"大部分光漏光后剩下的那一小撮"的集中显示。

所以正确的追问是：为什么一颗圆水滴——没有方向性，只发生部分反射——会让光在一个精确的 42° 方向上集中？为什么这个集中对波长敏感？

## 物理图像

1637 年，Descartes 在《方法论》的附录里做了一件今天看来非常现代的事：他不去问"彩虹为什么有颜色"，而是追踪一条光线在一个球形水滴里的命运。他问的是纯粹的几何问题——光从哪里进、从哪里出、往哪个方向走。

设一束平行阳光打到球形水滴上，入射角为 i（相对于入射点的法线，也就是该点到球心的连线）。光折射进入水滴，折射角 r 满足 Snell 定律 sin i = n sin r。在水滴背面发生一次（部分）内反射，然后从前表面再折射出去。追踪几何，出射光相对于原入射方向的总偏折角是

$$D(i) = 180° + 2i - 4r.$$

这条公式不起眼，但它是整个彩虹现象的骨架。**关键不在这个函数本身，而在它的导数。**

想象你把一束平行光的不同光线——打在水滴上缘、中部、接近边缘——都追踪一遍。每条光线的 i 不同，所以 D(i) 也不同。一般而言，入射角变化 Δi 会导致出射方向变化 ΔD。但 D(i) 作为一个函数，在某个特定的 i* 处有**极小值**：dD/di = 0。在这个极小点附近，D 对 i 几乎不变——也就是说，入射角在 i* 附近一小片区域里的光线，全部几乎朝同一个方向出射。这就是光的几何集中。

这是彩虹最深的物理：**焦散 (caustic) 是极值点的奇迹**。不是因为这个方向反射更多，不是因为这个方向不吸收——纯粹是因为 D(i) 的极值让大片入射光线"折叠"到同一个出射方向上。其他方向也有光出来，但稀散；只有极值方向，光强陡增。

从 dD/di = 2 - 4(dr/di) = 0 得到 dr/di = 1/2；对 Snell 定律 sin i = n sin r 两边微分，cos i di = n cos r dr，所以 cos i = (n/2) cos r。结合 Snell，经过一些三角代换，最终得到一个漂亮的闭式：

$$\cos^2 i^* = \frac{n^2 - 1}{3}.$$

水的 n ≈ 4/3，所以 cos²i* = (16/9 - 1)/3 = 7/27 ≈ 0.259，i* ≈ 59.4°。对应的 r* ≈ 40.2°，D_min ≈ 138°。彩虹方向（相对 antisolar point，也就是你影子头顶的方向）= 180° - 138° = **42°**。

这个 42° 是纯粹几何的产物。它不依赖水滴大小，只依赖 n。

现在回到三个谜题：

**为什么是弧形？** 水滴是球形的，关于入射光方向对称。所以对每一颗水滴来说，42° 不是一个方向，而是一个以 antisolar 方向为轴的**锥面**——锥顶半角 42°。空中无数颗水滴，每一颗都把它接收到的阳光散射成这样一个 42° 锥面。你只能看到落在"你的 42° 锥"上的那些水滴散出的光——这就是一条圆弧（被地平线切一半）。从飞机上、在瀑布边，地平线不挡，你就能看到整圆。

**为什么走不到它脚下？** 这个圆弧的轴线是你自己的 antisolar 方向——过你的眼睛、与太阳光相反的那条射线。你动一步，antisolar 方向跟着你动，整个 42° 锥面跟着你平移。彩虹不是挂在空间里的物体，而是**附着在观察者身上的方向图案**。两个人站在一起看同一道彩虹，其实看到的是两道几何上不同的彩虹——由两组不同的水滴散射出来，只不过方向重合得几乎完美。

**为什么有颜色？** n 对波长 λ 有微弱依赖（色散）。水在可见光中 n 从红光的 1.332 到紫光的 1.343，差 0.011，约 0.8%。这点微小差异通过 cos²i* = (n²-1)/3 放大到几何上——红光的 i* 略大、r* 略小、D_min 略小、彩虹角略大。结果：红色在 42.2°，紫色在 40.6°，差 **约 1.6°**。彩虹的可见"宽度"就是这个 1.6°——比太阳直径（0.5°）大三倍，所以颜色肉眼可分辨。

副虹（secondary rainbow）是光在水滴内部经历两次内反射后出来，D = 180° + 2i - 6r，极值给出 51°。两次反射把颜色顺序翻过来——副虹里红在内、紫在外。主虹外侧（> 42°）没有来自主虹机制的集中光，副虹内侧（< 51°）也没有来自副虹机制的集中光。两者中间这条 42°–51° 的环，**既不是 TIR 导致的暗，也不是吸收**，而是单纯的几何真空：没有哪种机制把光集中到这里。这就是 Alexander's dark band。

## 定量估算

把几何做扎实。水 n = 4/3（可见光平均值）。

**第一步：** 极值条件 cos²i* = (n² - 1)/3 = (16/9 - 1)/3 = 7/27。取平方根，cos i* ≈ 0.509，i* ≈ 59.4°。

**第二步：** Snell 定律给出 r*：sin r* = sin(59.4°)/n = 0.861/1.333 ≈ 0.646，r* ≈ 40.2°。

**第三步：** 代回 D = 180° + 2i - 4r = 180° + 118.8° - 160.8° = **138°**。彩虹角 = 180° - 138° = **42°**。三步算出来，没有查表。

**色散的量级估算。** 可见光谱从 700 nm（红）到 400 nm（紫），n 从 1.332 变到 1.343，Δn ≈ 0.011。对极值角 i* 求微分：2 cos i* (-sin i*) di* / (1) = (2n/3) dn，即 di*/dn = -(n/3) / (cos i* sin i*)。数值代入：di*/dn ≈ -(1.333/3) / (0.509 × 0.861) ≈ -1.01 rad，所以 Δi* ≈ -0.011 rad ≈ -0.63°。这是 i* 的变化；真正决定彩虹角的是 D_min 的变化。D = 180 + 2i - 4r，所以 ΔD = 2Δi - 4Δr。从 Snell 微分出 Δr，算到最后 ΔD ≈ -1.7°（红到紫 D_min 变小 1.7°，对应彩虹角从 42.2° 变到 40.6°）。**观测值 1.6°**——估算精度到 5% 以内，几何骨架经得起推敲。

**亮度量级。** 进入水滴的光，在第一次入射面 Fresnel 反射掉 ~2%，剩下 98% 进入；在水滴背面的内表面，入射角 40°（远低于临界角 48.6°），只有约 **4–6% 被反射回来**，其余 ~94% 漏掉；再从前表面折射出来，又损失 ~2%。主虹总效率 ≈ 0.98 × 0.05 × 0.98 ≈ **5%**。也就是说，你看到的彩虹只承载了 20 分之 1 的入射光能。这也是为什么彩虹只在阳光充足且背景偏暗（深色云层）时显眼——信噪比不够，它就被天空的散射光淹没。
